Pretendard

  • 국문/영문/숫자 모두 Pretendard를 표준 폰트로 사용한다.
  • 폰트 굵기는 Regular, SemiBold, Bold 3종류를 사용한다.
  • 본문 컨텐츠는 14px을 기준 폰트 사이즈로 정의한다.
  • 12px 미만의 폰트 사이즈는 권장하지 않는다.

스타일

  • font-family: 'Pretenard'; font-weight: 400; /* Regular */
  • font-family: 'Pretenard'; font-weight: 600; /* SemiBold */
  • font-family: 'Pretenard'; font-weight: 700; /* Bold */
